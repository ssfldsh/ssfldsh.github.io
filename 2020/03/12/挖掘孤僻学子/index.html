数据挖掘

朋友关系对挖掘：

实验要求：

1． 学会海量数据处理的方式方法。

2． 在时间序列中找到共现模式。

3． 研究高效处理海量数据的单机算法；尝试使用多机计算环境。

实验内容：

1．实验背景介绍

食堂是校园文化的主要组成部分之一，学生刷卡买饭的信息包含着有用的知识，希望你能挖掘出来。

从刷卡记录可以判断一个人在行为上是否是孤僻的，从而可以推断他/她在精神上是否疑似孤僻。

结伴的学生经常会有一同去一个餐厅吃饭的行为模式，你需要在大量数据记录中挖掘出好友，从而对那些没有好友的人进行疑似孤僻推断。

2．输入输出及具体要求

输入：文件“log.txt”，其中的每一行是一条刷卡记录。

例如，一行信息“2017-01-01_07:00:00,420,3,4”表示，在2017年1月1日的7点整，号码为420的学生去第3食堂的第4 POS机上刷卡买饭。

输出：每一个学生的好友。好友的定义：在给定的时间范围内，有超过比例R（例如1/4）的次数是在一个食堂吃饭的，而且两者的时间间隔前后不超过T分钟（例如5分钟）

例如，输出文件的一行如下：

“0#3,151,921”

这表示第0号学生有3个好友，分别是第3号、第151号和第921号学生。

具体要求：程序必须高效才能在有限的时间内完成任务。要在实验报告中把你的思路描述清楚。

实现方法:

创建Data类，包含以下五个变量

用arraylist创建data类的对象数组用于存储从文件中读入的信息

(一) Data类

package ex2;
public class Data {
    private String time;//存储从文件读入的时间字段
    private long newtime;//存储转换为毫秒后的时间
    private int number;//学号
    private int shiTang;//食堂号
    private int Pos;//pos机号
    public Data(String time, int number, int shiTang, int Pos) {
        this.time=time;
        this.number=number;
        this.shiTang=shiTang;
        this.Pos=Pos;
    }
    public String getTime() {
        return time;
    }
    public void setTime(String time) {
        this.time = time;
    }
    public int getNumber() {
        return number;
    }
    public void setNumber(int number) {
        this.number = number;
    }
    public int getPos() {
        return Pos;
    }
    public void setPos(int pos) {
        Pos = pos;
    }
    public int getShiTang() {
        return shiTang;
    }
    public void setShiTang(int shiTang) {
        this.shiTang = shiTang;
    }
    public long getNewtime() {
        return newtime;
    }
    public void setNewtime(long newtime) {
        this.newtime = newtime;
    }
    
}

                        shuang[record.get(i).getNumber()][record.get(j).getNumber()]++;
                        
                        shuang[record.get(j).getNumber()][record.get(i).getNumber()]++;
                    }
                }
            }

                if(dan[i]!=0) {

                            //System.out.println("the value of their relation:"+(float)shuang[i][j]/dan[i]);
