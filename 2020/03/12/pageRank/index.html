PageRank算法的应用：

实验要求：

1． 掌握PageRank算法的原理。

2． 学会编写PageRank的程序实现和应用。

实验内容：

实验背景介绍

PageRank是机器学习中重要的算法之一，在网络社区关系挖掘和相关系统中的挖掘至关重要，有许多版本变种和应用场景。

希望你能按照算法的基本原理，编写一个能够读取数据，并输出结果的系统。

注意，不要调用3行命令就实现的软件包。

输入输出及具体要求

输入：文件“linkArr.txt”，其中第一行是2个整数，n和m用英文逗号隔开。第1个整数n表示有多少个节点，第2个整数m表示有多少个关系对。

后面紧跟n行，每行表示一个链接关系，也用2个整数表示，第一个整数表示链接的发起节点，第2个整数表示链接的指向节点，中间用英文逗号隔开。

注意，节点序号是从0开始计数的。

当用户输入文件路径、多少代迭代后，就会产生各代迭代之后各节点的权重。

例如，如果链接文件路径是D:/linkArr.txt，其中的内容如下：

3, 2

0, 1

2, 1

则输入命令：

java PageRank “D:/linkArr.txt” 10 差异

我们要求：

（1） 算法初始化：每个节点分配概率是1/n。

（2） 算法的每一次迭代要根据上一次的概率比例进行。概率分配分为3部分：第一部分按照随机基础权重randomBasicWeight（此次实验设置为0.05）分配，也就是每次迭代每个节点首先获得randomBasicWeight/n的基础概率；第二部分是随机比例权重randomRatioWeight（此次实验设置为0.15），也就是每次迭代，各个节点按照上次的概率比例分配randomRatioWeight；第三部分：链接权重（此次实验设置为0.8），各节点按照上次迭代后的概率乘以链接权重，然后平均分配至它链接到的节点。

（3） 每迭代一次，都要进行归一化处理，使得各节点的概率之和在迭代之后保持为1。

实现方法：

package ex1;
import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;
import java.util.Scanner;

import ex2.Data;
public class pagerank {
    public static void main(String[] args) {
        try {
            //读入数据
            BufferedReader br=new BufferedReader(new FileReader("H:/linkArr.txt"));
            String s;
            int n=0;//节点数
            int m=0;//关系对数
            s=br.readLine();
            System.out.println("the value of n,m:"+s);
            String []strArr=s.split(",");
            n=Integer.parseInt(strArr[0]);
            m=Integer.parseInt(strArr[1]);
            Scanner input=new Scanner(System.in);
            int t;//迭代次数
            System.out.println("please input the value of iteration times--t:");
            t=input.nextInt();
            int [][]l=new int [100][100];//构造有向图l[i][j]==1说明节点i->j有一条边
            for(int i=0;i<100;i++) {
                for(int j=0;j<100;j++) {
                    l[i][j]=0;
                }
            }
            double p[]=new double[100];//p[i]存储节点权重
            for(int i=0;i<n;i++) {
                p[i]=1.0/n;
            }
            //构造有向图
            int lindex=0;
            int rindex=0;
            int leng[]=new int [n];//各个节点的出边数
            for(int i=0;i<n;i++) {
                leng[i]=0;
            }
            for(int i=0;i<m;i++) {
                s=br.readLine();
                String []strArr2=s.split(",");
                lindex=Integer.parseInt(strArr2[0]);
                rindex=Integer.parseInt(strArr2[1]);
                l[lindex][rindex]=1;
                leng[lindex]++;
            }
            double randomBasicWeight=0.05;
            double randomRatioWeight=0.15;
            double randomweight=1-randomBasicWeight-randomRatioWeight;
            //t次迭代，更新节点权重
            for(int i=0;i<t;i++) {
                double temp[]=new double [n]; 
                for(int j=0;j<n;j++) {
                    temp[j]=p[j];//temp[]存储上一代的节点权重值
                    p[j]=p[j]*randomRatioWeight+randomBasicWeight/n;
                }
                for(int j=0;j<n;j++) {//检查节点j的出边
                    for(int k=0;k<n;k++) {
                        if(l[j][k]==1) {
                            p[k]=p[k]+temp[j]*randomweight/leng[j];
                        }
                    }
                }
                double sum=0;
                //归一化
                for(int j=0;j<n;j++) {
                    sum=sum+p[j];
                }
                for(int j=0;j<n;j++) {
                    p[j]=p[j]/sum;
                }
                //输出结果
                System.out.println("the "+(i+1)+"th result:");
                for(int j=0;j<n;j++) {
                    System.out.println(p[j]);
                }
                //计算代间差异，当代间差异小于0.000001时，退出循环，停止迭代
                double cha=0.0;
                for(int j=0;j<n;j++) {
                    cha=Math.abs(p[j]-temp[j]);
                }
                //System.out.println("该代与上一代之间的代间差异:"+cha);
                if(cha<0.000001) break;
            }
        }catch(IOException e) {
            System.out.println(e.getMessage());
        } 
    }
}
